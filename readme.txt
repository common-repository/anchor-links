Auto add a summary in each article using his <h3> tags.

== Description ==

This plugin provides you to add a summary in the top of article, using article's <h3> tags. It also add a "Go to top page" link in each tags.

You can put all the anchor links inside the article wherever you want using a single function : ShowAnchorLinks().

Version française :

Ce plugin vous permet d'ajouter en haut de chacun de vos article un sommaire avec des liens pointant vers chacun des que vous aurez utilisé dans l'article. Il ajoute également un lien "Alelr au sommaire" devant chacun des intertitres .

Vous pouvez ajouter la liste de tous les liens ancres que contient votre article en utilisant la fonction ShowAnchorLinks().

== Frequently Asked Questions ==

= Quelles balises sont supportées? =

Pour le moment, seules les balises <h3>.

= Si je modifie mon article après publicatin, que se passe t-il? =
Toutes les modifications seront prises en compte, même les articles qui on été postés avant l'installation du plugin.
